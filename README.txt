سهيل - Sprint 03/07

الهدف:
إصلاح مشكلة سرعة الافتتاحية وعدم ظهور الصوت في Sprint 02.

التشغيل:
1) pip install -r requirements.txt
2) streamlit run app.py

ما تم في هذا السبرنت:
- ضبط مدة الافتتاحية إلى حوالي 3.3 ثواني.
- إضافة حد أدنى للظهور 3 ثواني حتى لا تختفي في أقل من نصف ثانية.
- تعديل ملف SVG نفسه لأن الأنيميشن الداخلي كان مدته قصيرة.
- توليد صوت افتتاح جديد أوضح وناعم داخل sounds/splash_soft.wav.
- رفع مستوى الصوت داخل الكود مع المحافظة على نعومته.
- إضافة fallback صوتي عبر WebAudio.
- إذا منع المتصفح الصوت التلقائي، يظهر تلميح خفيف: المس الشاشة للصوت.

ملاحظة مهمة:
Safari و iPhone وبعض المتصفحات تمنع الصوت التلقائي قبل أول لمسة. لذلك أضفنا تشغيل الصوت بعد أول لمس كخطة بديلة.

ملاحظات للسبرنت القادم:
- بعد تأكيدك أن التوقيت والصوت مناسبين، نقدر نخلي السبلاش يظهر أول مرة فقط بدل كل تحميل.
