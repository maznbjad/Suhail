# Sprint 01 — ضمان تغطية الفيزياء
## الهدف
بناء مصفوفة تغطية رسمية لكل كتب الفيزياء قبل إدخال التفاصيل، حتى نضمن لاحقًا أن كل تعريف وقانون ومثال وملاحظة مهمة موجودة داخل سهيل.
## التزام التصميم
لم يتم تغيير تصميم التطبيق. التحويل القادم سيستخدم نفس تصميم **ملخص ذكي** الحالي، وعلى مستوى الوحدة كما تم اعتماده.

1. استخراج كل التعاريف.
2. استخراج كل القوانين.
3. استخراج كل الأمثلة المحلولة.
4. إضافة الملاحظات والفخاخ المهمة.
5. ربط كل عنصر برقم الصفحة.
6. مراجعته داخل نفس تصميم الملخص الذكي.
