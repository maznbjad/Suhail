# إصلاح رجوع القائمة القديمة v28

## سبب المشكلة
الكاروسيل في الصفحة الرئيسية يعيد رسم محتوى الصفحة دوريًا. كانت القائمة الجديدة تُطبق كـ patch بعد الرسم، لكن عند إعادة الرسم كانت قائمة s18 القديمة تظهر مرة أخرى.

## الإصلاح
- تم استبدال مصدر القائمة داخل `renderHome()` نفسه بالقائمة الجديدة، وليس مجرد تعديل لاحق.
- تم جعل patch القائمة الجديدة احتياطيًا فقط إذا رجعت أي قائمة قديمة.
- أضيف MutationObserver وفحص خفيف للتأكد من ثبات القائمة بعد إعادة الرسم.
- بقيت التعديلات السابقة محفوظة:
  - الرئيسية فقط بدل الصفحة الرئيسية.
  - حذف أيقونة أعلى الهوم.
  - حذف بطاقة شرح تقسيم الفيزياء.
  - حذف عبارة الوحدات حسب الكتاب.
