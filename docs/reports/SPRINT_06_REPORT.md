# Sprint 06/12 — هيكل المسارات + بوصلة ملخص ذكي

## الهدف
تثبيت الهرم الصحيح للتطبيق قبل تنظيم كل الملخصات:

- الأقسام الرئيسية: قدرات / تحصيلي.
- داخل قدرات: كمي / لفظي.
- STEP محجوز لاحقًا كقسم رئيسي ثالث.

## التغييرات
- إعادة عرض بطاقات الصفحة الرئيسية لتظهر قدرات وتحصيلي فقط.
- عند اختيار قدرات تظهر كمي ولفظي كفروع.
- تحصيلي يفتح اختبار تحصيلي مباشرة حاليًا.
- إضافة ملف `data/navigation_map.json` كمرجع رسمي للهيكل.
- إضافة ملف `data/smart_summary_sources.json` لفهرسة ملفات الملخصات الخام.
- تحديث `data/smart_summaries.json` لدعم `track` و `subtrack`.
- تحويل صفحة ملخص ذكي إلى بوصلة بحث وفلاتر.
- دعم فلاتر: الكل / قدرات / كمي / لفظي / تحصيلي / رياضيات / فيزياء / كيمياء / أحياء.
- تضمين ملفات PDF الخام داخل `materials/raw_pdfs` حتى يبدأ تنظيمها لاحقًا.

## ملاحظات مهمة
- لم يتم تحويل كل PDFs إلى بطاقات نهائية في هذا السبرنت.
- بعض الملفات مصورة/مممسوحة وتحتاج تفريغ وتنظيم على دفعات.
- السبرنت القادم يبدأ بتحويل أول مادة إلى بطاقات منظمة داخل ملخص ذكي.
