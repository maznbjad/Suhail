# Sprint 03 — Transparent Suhail Character Fix

## التعديل
- إزالة خلفية شخصية سهيل عبر أصل PNG شفاف محسّن.
- تصغير الشخصية داخل كرت "متى أستخدمه؟".
- إزالة خلفية الحاوية الخاصة بالشخصية حتى تظهر الشخصية بدون بطاقة خلفية.
- تغيير عرض الصورة إلى contain لضمان ظهور الشخصية بشكل أنظف وبدون قص مزعج.

## السبب
النسخة السابقة كانت تعرض شخصية سهيل داخل حاوية بخلفية خضراء فاتحة، ومع أصل صورة غير مقصوص بالكامل بدت وكأن لها خلفية.
في هذا التحديث أصبحت الشخصية نفسها شفافة، والحاوية أيضًا بلا خلفية، مع تصغير الحجم قليلًا.
