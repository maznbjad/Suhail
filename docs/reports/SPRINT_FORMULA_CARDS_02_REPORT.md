# سهيل — سبرنت 2 لبطاقات القوانين

## المنجز
- تحويل عرض القوانين من نص عادي إلى تركيب رياضي بصري داخل HTML/CSS.
- دعم المؤشرات السفلية مثل `v_f` و`v_i` و`t_f` و`t_i`.
- دعم الأسس مثل `r^2` و`(Δt)^2`.
- دعم الكسور الحقيقية بخط كسر رأسي بدل علامة `/`، ومنها `Δd/Δt` و`1/2`.
- تصحيح بطاقة «ميل منحنى الموقع-الزمن» إلى صيغة القسمة: `v = Δd / Δt`.
- إكمال قائمة الرموز وعدم حصرها في ثلاثة رموز فقط.
- إضافة: المعطيات المطلوبة، المطلوب، انتبه، وتلميح سهيل داخل التفاصيل.
- جعل أقسام القرار تستجيب لشاشات الآيفون الصغيرة.

## فحوصات
- اجتاز الملف فحص Python (`py_compile`).
- لم تتغير آلية فتح وإغلاق البطاقات أو حفظ القانون.

## المطلوب اختباره على الجوال
1. قانون السرعة النهائية: ظهور `f` و`i` أسفل الحرف `v`.
2. قانون الإزاحة: ظهور `1/2` ككسر والأس `2` مرتفعًا.
3. ميل منحنى الموقع-الزمن: ظهور تغير الموقع فوق تغير الزمن، وليس بعلامة جمع.
4. قانون الجاذبية: ظهور `m₁m₂` فوق `r²` داخل كسر واضح.
