# Sprint 08/12 — Smart Summary Hierarchy

## الهدف
تثبيت هيكل الملخص الذكي كبوصلة منظمة، وليس قائمة مفتوحة.

## التغييرات
- حذف كرتي: إعادة صياغة، اختبار سريع.
- صفحة ملخص ذكي تبدأ بكرتين فقط: قدرات / تحصيلي.
- قدرات تعرض: كمي / لفظي.
- تحصيلي يعرض: رياضيات / كيمياء / فيزياء / أحياء.
- عند فتح مادة تظهر الوحدات.
- عند فتح وحدة تظهر بطاقات الملخص المتوفرة أو المصدر الخام إذا لم يتحول بعد.
- تحديث navigation_map إلى sprint_08.

## ملاحظات
- التحويل الكامل للملخصات الممسوحة إلى بطاقات سيتم على دفعات لاحقة.
- الاعتماد على الأيقونات والنصوص القصيرة مستمر.
