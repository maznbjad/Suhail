# سهيل — النسخة المنظمة Sprint 10

## التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

أو في ويندوز شغّل `run_suhail.bat`.

## التنظيم

- `app.py`: نقطة التشغيل الحالية للحفاظ على استقرار النسخة التجريبية.
- `assets/`: الصور، الهوية، الأيقونات، والأصوات.
- `data/`: الأسئلة، المستخدمون، الملخصات، وبنى المحتوى.
- `docs/`: التوثيق وتقارير السبرنتات السابقة.
- `scripts/`: أدوات فحص المشروع.
- `mobile/`: مكان نسخة React Native + Expo المستقبلية.
- `src/`: مكان فصل وحدات Python وAPI تدريجيًا.

## إدارة التطبيق

في حساب الأدمن أصبحت صفحة الحساب تعرض **إدارة التطبيق**، وتشمل:

1. إدارة المحتوى
   - إضافة سؤال جديد
   - مراجعة طلبات الأسئلة
   - تعديل الملخصات
2. إدارة الاشتراكات

محرر الملخصات يدعم التحصيلي ثم كتاب الفيزياء ثم الوحدة، ويتيح تعديل جميع عناصر الملخص وبطاقات القوانين القياسية. الحفظ في هذه النسخة محلي داخل المتصفح لاختبار الواجهة قبل نقل البيانات إلى الخادم.
