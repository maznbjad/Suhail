# Sprint 69 — Knowledge Linking Foundation

## الهدف
تحويل ملخصات الفيزياء المعتمدة إلى قاعدة معرفة دقيقة، وربط سؤال الفيزياء بالمعلومة التي يقيسها بدل ربطه بالدرس كاملًا.

## ما تم
- لم تُضف أي ملخصات جديدة؛ استُخدمت ملخصات الفيزياء الـ72 المعتمدة فقط.
- تقسيم الملخصات إلى 936 كتلة معرفة ثابتة المعرف.
- أنواع الكتل: الفكرة الجوهرية، التعريف، القانون، العلاقة، الفخ، تلميح سهيل، المثال، والمقارنة.
- ربط 600 سؤال فيزياء بالمُلخص والدرس والكتلة الدقيقة.
- إضافة البيانات المخفية للسؤال: `summary_id`, `summary_block_id`, `misconception_block_id` وبيانات العرض المختصرة.
- تحديث JSON وقاعدة SQLite ومخططات محرر السؤال والملخص.
- إظهار رابط «راجع المعلومة» بعد الإجابة فقط عندما يكون خيار عرض النتيجة مفعّلًا.
- تطوير الأسئلة المحفوظة لتحتفظ برقم السؤال والإجابة والتوضيح والكتلة المرتبطة، مع انتقال مباشر إليها داخل الملخص.
- دعم الوضع الداكن للبطاقات الجديدة.

## التغطية الحالية
- 72 ملخص فيزياء جاهزة للربط.
- 600 سؤال فيزياء مرتبط بنسبة 100%.
- الأسئلة الحالية تغطي 22 ملخصًا فعليًا؛ 50 ملخصًا لا يوجد في بنك الفيزياء الحالي سؤال يقيسها حتى الآن. لم تُنشأ أسئلة وهمية لرفع التغطية.

## حدود المرحلة
هذه المرحلة تبني الربط الهيكلي الدقيق ولا تدّعي إتمام مراجعة تربوية بشرية لكل سطر من الملخصات. كما لم تُنشأ ملخصات للكيمياء أو الرياضيات أو الأحياء وعلم البيئة التزامًا بقرار عدم إضافة ملخصات تلقائيًا.
