# سهيل — Sprint 108

نسخة محدثة من تطبيق سهيل بنظام **ملف PDF واحد لكل كتاب** ووضع قراءة مخصص للطالب.

## الملخصات المركبة

- فيزياء 1: 18 صفحة.
- فيزياء 2: 13 صفحة.
- فيزياء 3 - الفصل الأول: 22 صفحة.

جميع الكتب المركبة تتضمن طبقة نص OCR قابلة للتحديد، مع تظليل محفوظ بأربعة ألوان.

## تجربة القراءة

- يفتح الكتاب بملء شاشة الجوال.
- تختفي الترويسة والقائمة السفلية وجميع عناصر التطبيق.
- يبقى زر العودة فقط في الجهة اليمنى، بخلفية شفافة 50٪.
- زر العودة يحجز مساحة مستقلة ولا يتداخل مع العناوين.
- صفحات الكتاب متصلة رأسيًا دون فراغات بين الصفحات.
- يحفظ التطبيق آخر صفحة والتظليلات لكل كتاب بشكل مستقل.
- الصور تستخدم التحميل الكسول وفك الترميز غير المتزامن لتحسين الأداء.

## إطار الآيفون والوضع الفعلي

- على الكمبيوتر يظهر التطبيق كاملًا داخل إطار آيفون 390 × 844.
- الوضع الداكن محصور داخل شاشة الجهاز، والخلفية الخارجية فاتحة.
- على الجوال الحقيقي يختفي الإطار التجريبي والـDynamic Island وشريط الحالة الوهمي، ويملأ التطبيق الشاشة الفعلية مع احترام Safe Areas.

## التشغيل على Windows

1. فك ضغط الملف في مجلد جديد.
2. شغّل `run_suhail.bat`.
3. يثبت الملف المتطلبات تلقائيًا عند أول تشغيل.
4. افتح الرابط الظاهر في نافذة التشغيل.

## إدارة الملخصات

المسار:

`إدارة التطبيق ← إدارة المحتوى ← تعديل الملخصات`

تعرض الصفحة حالة كل كتاب وعدد صفحاته، مع خانة رفع PDF لكل كتاب.

## حالة رفع Apple

طبقة التصميم وتجربة المستخدم مجهزة للانتقال إلى تطبيق iOS، لكن هذا المستودع الحالي ما زال نموذج Python/Streamlit وليس ملف IPA. تفاصيل الخطوات المتبقية موجودة في:

`mobile/APP_STORE_READINESS.md`

الإصدار: `108.0.0`
