# Sprint 07/12 - تحسين البوصلة الرئيسية والملخص الذكي

## ما تم
- تحسين توزيع كرتي الصفحة الرئيسية: قدرات / تحصيلي.
- جعل البطاقات أعرض، والمسافة بينهما أوضح، والنصوص أقل.
- عند الضغط على قدرات تظهر الفروع: كمي / لفظي.
- عند الضغط على تحصيلي تظهر المواد: رياضيات / فيزياء / كيمياء / أحياء.
- تحصيلي الآن لا يفتح الاختبار مباشرة؛ يوجه الطالب للملخص الذكي حسب المادة.
- إضافة مؤشرات صغيرة داخل الملخص الذكي: عدد البطاقات، المصادر، والروابط.
- إضافة أول دفعة بطاقات ذكية تربط بين التحصيلي والقدرات:
  - التناسب الطردي والعكسي
  - التناظر اللفظي
  - التبرير والاستنتاج
  - المادة وتغيراتها
  - خصائص الحياة

## ملاحظات
- لم يتم توليد أي صور.
- خط Tajawal يعمل عبر CSS import مع fallback نظامي.
- ملفات PDF الخام بقيت داخل materials/raw_pdfs للتنظيم على دفعات.

## ينتقل للسبرنت القادم
- تحويل المزيد من ملخصات الرياضيات والكيمياء والأحياء إلى بطاقات ذكية.
- ربط نتيجة الاختبار ببطاقات الملخص الذكي.
- تحسين صفحة المادة داخل الملخص الذكي.
