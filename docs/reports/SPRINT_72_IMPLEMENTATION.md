# Sprint 72 — الربط الدقيق والتوضيح التعليمي

## الربط بين السؤال والملخص

- أصبح الرابط يعتمد على `summary_id` و`summary_block_id` معًا.
- زر المرجع يعرض المسار: الكتاب/المرحلة، الوحدة، الدرس، ثم الفقرة المعرفية.
- عند فتح الملخص تظهر الفقرة المرتبطة أعلى الوحدة ويُميّز الدرس المقصود.
- أُصلح فلتر الأسئلة المرتبطة ليطابق الوحدة الحالية فقط، بدل قبول أي وحدة لها اسم.
- أُلغيت الروابط التخمنية. السؤال الذي لا يملك ملخصًا منشورًا مطابقًا لا يظهر له رابط مضلل.
- جرى التحقق من 600 رابط فيزياء، وجميعها تشير إلى ملخص وكتلة معرفية صالحين.

## سياسة التوضيح

- سهل: لا يظهر شرح زائد؛ تبقى الإجابة متاحة عند الحاجة.
- متوسط: شرح موجز يوضح قاعدة الاختيار وطريقة التطبيق والفخ القريب عند الحاجة.
- صعب: ملخص للحل، خطوات مرتبة، وفخ شائع أو نقطة تحقق.
- بُنيت التوضيحات من النصوص التحريرية الأصلية، ثم دُعمت بأدلة مهارية مخصصة بدل العبارات العامة.

## التحقق

- 5,400 سؤال.
- 1,968 سؤالًا سهلًا بوضع `none`.
- 3,137 سؤالًا متوسطًا بوضع `brief`.
- 295 سؤالًا صعبًا بوضع `full`.
- لا يوجد سؤال متوسط أو صعب بلا توضيح.
- لا يوجد سؤال سهل يحمل شرحًا ظاهرًا.
- اختبار تفاعلي للرابط والنافذة وفتح موضع الملخص وبطاقات النتيجة: ناجح دون أخطاء JavaScript.
- اختبار API: الإصدار 72.0.0، قاعدة البيانات متاحة، و72 ملخصًا مسترجعًا.
