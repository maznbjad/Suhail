# سهيل — Sprint 103

نسخة تجريبية محدثة لنظام ملخصات PDF، مع تركيب **ملخص فيزياء 1 — 18 صفحة** داخل التطبيق.

## التشغيل على Windows

1. فك ضغط الملف كاملًا في مجلد جديد.
2. شغّل `run_suhail.bat`.
3. في التشغيل الأول تُثبت المتطلبات تلقائيًا.
4. افتح الرابط الظاهر في نافذة التشغيل، ويمكن فتحه من الجوال على شبكة Wi‑Fi نفسها.

## تحديث قارئ الملخصات

- يفتح **فيزياء 1** بوضع قراءة كامل يغطي واجهة التطبيق.
- تختفي القوائم السفلية، الترويسة، رقم الصفحة وأزرار الأدوات بالكامل.
- يبقى زر العودة فقط، بخلفية شفافة بنسبة 50٪.
- أضيفت طبقة OCR فوق صفحات الملف المصورة، لذلك يمكن تحديد الكلمات والأسطر.
- بعد تحديد النص تظهر لوحة ألوان صغيرة مؤقتًا فقط.
- ألوان التظليل: أصفر، أخضر، أزرق ووردي.
- يحفظ التظليل وموضع القراءة على الجهاز.
- بقي التصميم الأصلي للـPDF كما هو دون إعادة تنسيق الصفحات.

## إدارة المحتوى

المسار:

`إدارة التطبيق ← إدارة المحتوى ← تعديل الملخصات`

تعرض الصفحة ملفًا مستقلًا لكل كتاب، مع توضيح أن ملف فيزياء 1 مركب حاليًا مع طبقة نص.

