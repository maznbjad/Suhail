# Suhail Sprint 02/07 Report

## الهدف
ضبط شاشة الافتتاحية المتحركة وإضافة صوت افتتاح ناعم بدون تغيير منطق الاختبار.

## المنجز
- إضافة ملف صوت محلي: `sounds/splash_soft.wav`.
- ربط الصوت بشاشة الافتتاحية داخل `app.py`.
- إضافة محاولة تشغيل تلقائي للصوت عند بداية السبلاش.
- إضافة fallback صوتي بالـ WebAudio لو تعذر تشغيل ملف الصوت.
- إضافة إعادة محاولة للصوت عند أول تفاعل من المستخدم إذا منعه المتصفح.
- إضافة زر `تخطي` للسبلاش.
- ضبط مدة الافتتاحية إلى حوالي 2.7 ثانية.
- منع تكرار الافتتاحية بعد أول ظهور عبر `localStorage`.

## ملاحظات مهمة
- بعض المتصفحات، خصوصًا iOS/Safari وChrome أحيانًا، تمنع تشغيل الصوت تلقائيًا قبل أول لمسة من المستخدم. الكود يحاول التشغيل مباشرة، وإذا تم المنع يعيد المحاولة عند أول لمس/نقر.
- إذا أردت إعادة تجربة السبلاش بعد ظهوره أول مرة، امسح بيانات الموقع من المتصفح أو نفذ في Console:

```js
localStorage.removeItem('suhail_splash_seen_v2')
```

## ما لم يتم لمسه
- منطق الأسئلة والاختبار.
- صفحة النتيجة.
- لوحة الأدمن.
- بنك الأسئلة.

## اختبارات مقترحة
1. شغل التطبيق وتأكد أن الافتتاحية تظهر.
2. تأكد أن الصوت ناعم وغير مزعج.
3. اضغط زر تخطي وتأكد أنه ينقلك مباشرة.
4. أعد تحديث الصفحة وتأكد أن الافتتاحية لا تتكرر.
5. جرّب على الجوال، خصوصًا الصوت بعد أول لمسة.
