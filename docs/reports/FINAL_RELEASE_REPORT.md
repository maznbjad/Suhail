# Suhail Final Sprint Bundle

هذه النسخة تجمع Sprint 01 إلى Sprint 12.

أهم ما في النسخة:
- الصفحة الرئيسية: مدخل اختبار فقط.
- ملخص ذكي: بوصلة مستقلة للملخصات.
- القدرات: كمي / لفظي كفروع داخل قدرات.
- التحصيلي: رياضيات / كيمياء / فيزياء / أحياء.
- تصميم أخف وأقل نصوصًا.
- خط Tajawal عبر CSS import/fallback.
- نقاط XP ورتب وتحديات أصدقاء كتجربة محلية.
- إعداداتي كأدمن كتجربة محلية.

التشغيل:
```bash
pip install -r requirements.txt
streamlit run app.py
```
