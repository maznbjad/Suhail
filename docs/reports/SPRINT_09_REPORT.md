# Sprint 09/12 — فصل الاختبار عن ملخص ذكي

## الهدف
إصلاح الخلط بين الصفحة الرئيسية الخاصة بالاختبارات وبين بوابة ملخص ذكي.

## التغييرات
- الصفحة الرئيسية أصبحت للاختبار فقط.
- كرت قدرات يفتح اختيار كمي/لفظي للاختبار.
- كرت تحصيلي يفتح إعداد اختبار تحصيلي مباشرة، ولا يدخل إلى مواد الملخص.
- ملخص ذكي أصبح بوابة مستقلة:
  - قدرات / تحصيلي
  - داخل قدرات: كمي / لفظي
  - داخل تحصيلي: رياضيات / كيمياء / فيزياء / أحياء
  - داخل المادة: الوحدات
  - داخل الوحدة: بطاقات الملخص أو مصدر خام قيد التنظيم
- إلغاء ظهور الواجهة القديمة للملخص عند الدخول من كيمياء أو فيزياء.
- حذف/إخفاء كروت أدوات غير مطلوبة عند وجودها: إعادة صياغة، اختبار سريع.

## ملاحظات
الملخصات الخام المصورة ما زالت قيد التحويل إلى بطاقات مختصرة في السبرنتات القادمة.
