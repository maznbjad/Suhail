# Suhail — Sprint 101

نسخة سهيل النهائية لمسار الملخصات البصرية للفيزياء والكيمياء.

## التشغيل على Windows

1. فك ضغط الملف كاملًا في مجلد جديد.
2. شغّل `run_suhail.bat`.
3. في التشغيل الأول ستُثبت المتطلبات تلقائيًا.
4. سيظهر رابط الكمبيوتر ورابط الجوال على شبكة Wi-Fi نفسها.

## محتوى Sprint 101

- 72 ملخص فيزياء بالقالب البصري كامل الشاشة.
- 83 ملخص كيمياء موزعة على: كيمياء 1، كيمياء 2-1، كيمياء 2-2، كيمياء 3.
- 155 ملخصًا منشورًا إجمالًا.
- 67 رسمًا علميًا أصليًا بصيغتي SVG وWebP، بجودة WebP متوسطة محسنة للأداء.
- ربط 600/600 سؤال كيمياء بالكتاب والفصل والدرس والفقرة المعرفية.
- الإبقاء على روابط أسئلة الفيزياء السابقة.
- صفحة الدرس لا تعرض شريطًا سفليًا أو أيقونات جانبية؛ يظهر زر الرجوع فقط في الأعلى.
- الأحياء وعلم البيئة منفصلان داخل قائمة الملخصات، مع بقاء قسم الاختبار مشتركًا.

## بنية صفحة الملخص

- الفكرة الأساسية ولماذا يحدث؟
- التعريفات ولا تخلط.
- الفخ الشائع وكيف يأتي في التحصيلي؟
- رسم علمي أصلي مخصص للمفهوم.
- قانون أو قاعدة بعرض كامل.
- ثلاثة أمثلة متدرجة.
- خمسة أسئلة إتقان.
- خلاصة مركزة.
