# سبرنت 01 — بطاقة القوانين الذكية المضغوطة

## المنجز

- استبدال العرض الكبير للقانون ببطاقة عملية ومضغوطة.
- إظهار أول قانون مفتوحًا افتراضيًا، وبقية القوانين مغلقة.
- فتح وإغلاق التفاصيل من السهم العلوي أو زر «عرض التفاصيل».
- جعل المعادلة أقوى عنصر بصري داخل البطاقة.
- إبقاء «متى أستخدمه؟» ظاهرة حتى في الحالة المغلقة.
- عرض الرموز كشارات صغيرة، ثم عرض معناها داخل التفاصيل.
- إضافة زر حفظ مستقل لكل قانون.
- حفظ القوانين المختارة محليًا في `localStorage` بالمفتاح:
  `suhail_saved_laws_v1`.
- تحديث ملف شعار سهيل بالشعار المرسل من المستخدم.
- حفظ وضعية الإشارة لشخصية سهيل كمرجع بصري للسبرنت التالي.

## نطاق السبرنت الحالي

يعمل التصميم الجديد في الصفحات التي كانت تستخدم قالب القوانين المطور سابقًا:

- فيزياء 1
- فيزياء 2

## مؤجل إلى السبرنت 02

- المعطيات المطلوبة والمطلوب لكل قانون.
- صندوق «انتبه» بمحتوى مخصص.
- «تلميح سهيل» مع الشخصية الفعلية.
- إظهار جميع الرموز والتحقق من معناها بدل الاكتفاء بالبيانات الحالية.
- تعميم القالب على فيزياء 3-1 وفيزياء 3-2.
- المعالجة النهائية للكسور والقوانين الطويلة جدًا.

## الفحص

- اجتاز `app.py` فحص Python دون أخطاء.
- اجتاز JavaScript المضمن فحص الصياغة باستخدام Node.js.
- لم يتم تغيير بنية بيانات الملخصات أو الأسئلة.
