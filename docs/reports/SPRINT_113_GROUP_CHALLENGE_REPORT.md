# Sprint 113 — Group Challenge

- 2–10 players, including the owner.
- Fixed 30 seconds per question.
- Server-authoritative question timer and answer validation.
- First correct answer wins the point and advances the room immediately.
- Wrong answers cannot be retried for the same question.
- Live scoreboard, final ranking and time-management feedback.
- REST polling is used in this Streamlit prototype; the native production client can replace polling with WebSocket events without changing the scoring model.
