# Sprint 55 — تقرير صفحة الحساب

## الهدف

اعتماد تقسيم صفحة الحساب المرجعي: عنوان مركزي، ملف طالب مختصر، مجموعة إعدادات واحدة ذات صفوف وفواصل، ثم إصدار التطبيق وبياناته وزر الخروج.

## التنفيذ

- بطاقة ملف مختصرة تعرض الاسم، الاختبارات المختارة، مسار القدرات، وكود الصداقة.
- قائمة موحدة تضم: رحلتي التعليمية، تحديد المستوى، الوضع الداكن، الإشعارات، الأصدقاء والتحديات، الأسئلة المحفوظة، عن سهيل، تواصل معنا، الخصوصية، الشروط، والأسئلة الشائعة.
- أيقونات SVG خطية متجانسة، مع سهم انتقال موحد في الطرف المقابل.
- الوضع الداكن مفتاح مباشر داخل القائمة.
- صفحات داخلية للإشعارات، التواصل، عن سهيل، الخصوصية، الشروط والأسئلة الشائعة.
- مجموعة إدارة مستقلة تظهر للأدمن فقط.
- رقم الإصدار `V.1.0.55` وتذييل هوية سهيل وزر تسجيل الخروج.

## حدود قبل النشر

- نصوص الخصوصية والشروط داخل هذه الحزمة نصوص تصميم أولية وليست صياغة قانونية نهائية.
- إرسال رسائل التواصل يحفظها محليًا إلى أن يتم ربط مركز دعم بالخادم.
- الإشعارات بعد إغلاق التطبيق تتطلب Expo/APNs.
