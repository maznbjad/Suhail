# بطاقات القوانين — سبرنت 07

## المرجع المعتمد
تم اعتماد تصميم قوانين وحدة **تمثيل الحركة** كمرجع موحد لجميع وحدات الفيزياء.

## التعديلات
- جميع مراحل فيزياء 1، فيزياء 2، فيزياء 3-1، وفيزياء 3-2 تستخدم الآن المكوّن نفسه `s17LawCard`.
- إزالة العرض القديم المختصر الذي كان يظهر في الوحدات الأخرى.
- إزالة CSS القديم الخاص ببطاقات القوانين السابقة لمنع أي اختلاف في الخط أو اللون أو المسافات.
- توحيد MathML، مقاس القانون 20px، ألوان الهوية، الحواف، المسافات، الحفظ، الفتح والإغلاق، الرموز، وتلميح سهيل.

## النتيجة
لا يوجد مسار عرض بديل للقوانين حسب المرحلة؛ كل قانون يمر عبر مكوّن موحد مبني على تصميم تمثيل الحركة.
